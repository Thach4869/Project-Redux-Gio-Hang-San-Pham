# Project-Redux-Gio-Hang-San-Pham
Trong bài tập này, bạn sẽ xây dựng một ứng dụng giỏ hàng sử dụng React kết hợp Redux để quản lý toàn bộ trạng thái sản phẩm và giỏ hàng. Ứng dụng mô phỏng trải nghiệm mua sắm cơ bản, nơi người dùng có thể xem danh sách sản phẩm, thêm vào giỏ, thay đổi số lượng và xem tổng tiền thanh toán.
